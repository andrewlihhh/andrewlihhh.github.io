### 4.1 住宿据点

- id：hotel
- 编号展示：住
- Day 标签：Hotel
- 标题：汤口镇山脚住宿据点
- 经纬度：30.1316,118.1692
- 时间说明：抵达日 / 上下山衔接
- 区域说明：住宿据点
- 摘要：住在汤口镇的好处是换乘、上山、下山都顺，尤其适合第一次来黄山的人。
- 历史 / 看点：这里不是景点型住宿，而是典型的黄山山脚中转据点。很多黄山攻略都会把汤口镇作为第一晚和最后一晚的落脚点，因为它离南大门、换乘中心和常用索道入口都近。
- 吃什么更顺路：推荐美食：毛豆腐、臭鳜鱼、茶干、清炒时蔬。不能食用：徽州烧饼常见猪肉馅、刀板香、火腿炖笋。
- 拍照建议：更适合拍“出发感”而不是目的地大片，比如清晨整装出发、雨后街道、山脚云雾。
- 对应图片（可选）：无

## 5. 三日游 / 多日游卡片

> 如果不是三日游，可以写成 Day 1 / Day 2 / Day 3 / Day 4……

对每一天重复下面格式：

```md
### Day
- day：Day 1
- 标题：上山主景线
- 副标题：第一次去黄山最稳的起手式，就是从汤口镇轻装上山，把北海和光明顶主线走顺，晚上住山上。
- 行程安排：
  - 前一晚住汤口镇，早上尽量早点去换乘和排索道
  - 从云谷索道上山，优先把始信峰、北海一线和光明顶串起来
  - 下午把天气最好的时间留给光明顶和沿途观景平台
  - 晚上住山上，第二天再看要不要追日出、云海和西海大峡谷
- 推荐美食：热豆浆、茶叶蛋、素包、山顶素面、坚果、能量棒
- 不能食用：猪肉包、香肠、肉酱面、肉松面包
- 交通建议：这一天建议尽量轻装，换洗衣物和大件行李留在山下住宿点。
```

```md
### Day
- day：Day 2
- 标题：精华峡谷 + 宏村线
- 副标题：第二天的上限不在景点数量，而在天气和体力。状态好就冲西海大峡谷，状态一般就收成迎客松 + 宏村。
- 行程安排：
  - 清晨先看天气，如果有云海或日出窗口，优先把最好的光线留给山上
  - 体力够、天气稳，再走西海大峡谷；如果下雨或膝盖压力大，就直接收成主线下山
  - 中午往迎客松、玉屏线方向收束
  - 下午下山，傍晚把宏村留给月沼、南湖和巷子里的慢节奏
- 推荐美食：毛豆腐、臭鳜鱼、五城茶干、笋干烧豆腐、酸奶冰品
- 不能食用：刀板香、火腿炖笋、猪肉馄饨、猪肉串
- 交通建议：西海大峡谷对膝盖和体力要求比较高，天气不好时可以直接收缩路线。
```

```md
### Day
- day：Day 3
- 标题：西递 + 屯溪老街收尾线
- 副标题：最后一天不要再拼强度，重点是把徽州古村和老街气质慢慢收进来，让这趟黄山有个完整结尾。
- 行程安排：
  - 上午去西递看牌坊、巷道和民居细节，也可以换成呈坎或西溪南
  - 中午不要安排太满，留一点休息和整理照片的时间
  - 下午到傍晚去屯溪老街慢逛、拍照和收尾
  - 如果还有精力，晚上补一段黎阳 in 巷；如果没有，就把这天收成一个轻松返程日
- 推荐美食：徽州豆腐、茶干、臭鳜鱼、山笋、清炒时蔬、蓝莓酸奶
- 不能食用：猪肉馅黄山烧饼、刀板香、火腿、猪肉小馄饨
- 交通建议：这一天不需要赶，重点是让旅行有个从容的结束。
```

## 8. 结尾说明 / 补充提示

### 卡片 1

- 标题：第一次去黄山，住山上一晚更值
- 正文：很多公开攻略都会提到，黄山如果只是当天上下山，容易把行程走成纯体力测试。住山上一晚，会让光线、天气和山景层次都更从容。

### 卡片 2

- 标题：山上天气比山下更重要
- 正文：黄山真正决定体验的，不只是路线，而是天气。云海、雾、日出、日落都会直接改变观感，所以行程最好保留一点弹性。

### 卡片 3

- 标题：古村不是附赠，是黄山旅行的另一半
- 正文：如果只看山，你会记住黄山的壮阔；但把宏村、西递和老街一起放进来，这趟旅行才会从“风景”变成“完整的徽州体验”。
