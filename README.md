# andrewlihhh.github.io Astro 版本

这是一个基于 Astro 的静态图文站，当前内容已经改造成“哈尔滨旅拍手册”单页。

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物在 `dist/`。

## Cloudflare Pages 建议配置

- Framework preset: `Astro`
- Build command: `npm run build`
- Build output directory: `dist`
- Production branch: 你连接到 GitHub 的主分支
